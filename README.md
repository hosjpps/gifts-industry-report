# Gifts Industry — дизайн-документация бренда

Single source of truth для дизайна премиального подарочного бренда на Wildberries: дизайн-система, нарратив карточки товара, бренд-лендинг и студийные шоты.

## Возможности

- **Дизайн-бриф** (`design-brief.md`) — самостоятельный пакет: позиционирование, палитра, шрифты, типографические правила, нарратив 10 слайдов карточки WB, готовый копирайт и anti-patterns с обоснованием.
- **Бренд-лендинг** (`index.html`) — одностраничник в чёрно-золотом стиле: отчёт о позиционировании и стратегии продвижения, интерактивные диаграммы (Chart.js), адаптивная вёрстка. Деплоится через Vercel.
- **Студийные шоты** (`master-shots/`) — финальные продуктовые фото набора, на которые накатывается типографика слайдов.
- **Изображения бренда** (`*.webp`) — AI-модели и продуктовые шоты ложки и кружки.
- **Готовая дизайн-система** — единая палитра, шрифты и токены, переиспользуемые для карточек, рекламных креативов и соцсетей.

## Стек

- HTML / CSS (статический лендинг, адаптивная вёрстка)
- [Chart.js](https://www.chartjs.org/) — интерактивные диаграммы
- Google Fonts (Playfair Display, Inter, Cormorant Garamond)
- Vercel — деплой лендинга
- Markdown — дизайн-документация

## Структура

| Файл / папка | Что |
|--------------|-----|
| [`design-brief.md`](design-brief.md) | **Главный документ.** Single source of truth: позиционирование, дизайн-система, нарратив 10 слайдов карточки WB, копирайт, anti-patterns |
| [`index.html`](index.html) | Лендинг бренда (одностраничник, чёрно-золотой стиль, отчёт о позиционировании). Живая версия деплоится через Vercel |
| `master-shots/set-1/` | Финальные студийные шоты товара набора 1 (с ложкой) |
| `*.webp` | Изображения бренда: AI-модели (мужчина / женщина) + продуктовые шоты ложки и кружки |

## Запуск

Лендинг — статический, без сборки. Открыть локально:

```bash
open index.html
```

Либо поднять любой статический сервер:

```bash
python3 -m http.server 8000
# затем открыть http://localhost:8000
```

## Для Claude Design / других AI

При работе над визуалом **читать в первую очередь:**

1. `design-brief.md` — палитра, шрифты, нарратив 10 слайдов, готовый копирайт, anti-patterns с пометкой откуда они выведены
2. `index.html` — как уже воплощён бренд: чёрно-золотая палитра, серифные заголовки, тон сообщений
3. `master-shots/set-1/` — чистые фото товара; накатывать на них типографику и подписи из brief

**Что НЕ нужно искать в репо:** маркетинговую стратегию, исследование рынка, видео-сценарии — этих документов здесь нет.
