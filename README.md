# Gifts Industry — design assets

Репозиторий с дизайн-документацией бренда **Gifts Industry** (премиальные подарочные наборы на Wildberries).

## Что внутри

| Файл / папка | Что |
|--------------|-----|
| [`design-brief.md`](design-brief.md) | **Главный документ.** Single source of truth: позиционирование, дизайн-система, нарратив 10 слайдов карточки WB, копирайт, anti-patterns |
| [`index.html`](index.html) | Лендинг бренда (одностраничник, чёрно-золотой стиль, отчёт о позиционировании). Живая версия деплоится через Vercel |
| `master-shots/set-1/` | Финальные студийные шоты товара набора 1 (с ложкой). Загружаются по мере готовности |
| `master-shots/set-2/` (будет) | То же для набора 2 (с кружкой) |
| `*.webp` | Изображения бренда: AI-модели (мужчина / женщина) + продуктовые шоты ложки и кружки |

## Для Claude Design / других AI

При работе над визуалом **читать в первую очередь:**

1. `design-brief.md` — там палитра, шрифты, нарратив 10 слайдов, готовый копирайт, anti-patterns с пометкой откуда они выведены
2. `index.html` — посмотреть как уже воплощён бренд: чёрно-золотая палитра, серифные заголовки, тон сообщений
3. `master-shots/set-1/*.jpg` — чистые фото товара. Накатывать на них типографику и подписи из brief

**Что НЕ нужно искать в репо:** маркетинговую стратегию, исследование рынка, видео-сценарии — этих документов здесь нет, они в локальном архиве и для дизайн-задачи не нужны.
